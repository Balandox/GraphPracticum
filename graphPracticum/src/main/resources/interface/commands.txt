
Список доступных команд:
1. generate        перейти к меню генерации заданий.
2. files -add      указать файлы для записи условий заданий и ответов.
3. files -show     посмотреть файлы для записи условий заданий и ответов.
4. help            посмотреть список доступных команд.
5. clear           очистить консоль.
6. exit            завершение работы.
